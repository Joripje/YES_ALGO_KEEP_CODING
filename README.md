## :round_pushpin: Rules

### Algorithm :wave:

- ####  Path: /Algorithm/{Platform}_{Qustion_Num}.py

- Based on `Python`

- Platform 코드는 백준: `boj`, 프로그래머스: `ps`, 리트코드: `lc` 로 통일
- 업로드 후 README의 Records 항목을 추가하고 Reports 에 경로를 추가할 것
